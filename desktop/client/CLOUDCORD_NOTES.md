# CloudCord Notes

Imported Sincord commit: `2f555631e72c92d5cdb4d8763be2c3ba0e0debf1`.

CloudCord Desktop uses the in-repo desktop client mod runtime built from this source tree. It does not load `dist/cc.js`, and it does not use the iOS or Android React Native runtime.

Desktop identity:

- Product: CloudCord
- Runtime: CloudCord Desktop
- Version: 0.1.0
- Repository: https://github.com/xohus/cloudcord

The old homemade desktop runtime and Electron installer were removed and replaced by this Sincord-based source tree.
